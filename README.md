## 나만의 포트폴리오 웹사이트 만들기

- Notion API를 활용하여 만들게된 포트폴리오 웹페이지 입니다. 나의 노션과 연동을시켜, 노션 데이터베이스에 자료 추가시, 포르폴리오페이지에 자동 업로드 가능합니다.

## 💻 기술스택

- Nextjs
- Javascript
- Tailwind CSS
- Notion API
- Lottie Animation
- vercal deplay
